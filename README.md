
# Información de la Exposición y Práctica del Segundo Corte

Repositorio que contiene el informe de la exposición grupal correspondiente al **segundo corte** de la asignatura **Ingeniería de Requisitos**, donde se abordaron distintas herramientas CASE, como **StarUML, Umple Online, Eclipse Papyrus, BOUML, Modelio y Visual Paradigm**, utilizadas para el modelado UML y la generación automática de código.

# 🧩 Temas cubiertos en el informe

- Conceptos de **MDD (Model-Driven Development)**, **MDE (Model-Driven Engineering)** y **MDA (Model-Driven Architecture)**.
- Transformaciones **M2M (Model to Model)** y **M2T (Model to Text)**.
- Comparación de herramientas CASE:
  - StarUML
  - Umple Online
  - Eclipse Papyrus + Acceleo
  - BOUML
  - Modelio
  - Visual Paradigm
- Ingeniería directa.
- Ingeniería inversa.
- Ingeniería bidireccional.
- Prácticas de generación automática de código a partir de diagramas UML.
